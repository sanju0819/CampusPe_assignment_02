## 📌 About The Project

This repository contains all solutions developed as part of the **CampusPE Python Basics Assignment (Modules 1–6)**.

It includes **20 programming problems** covering Python fundamentals, conditionals, loops, functions, string manipulation, mathematical computations, and mini projects like **ATM Simulator** and **Number Guessing Game**.

All programs are written in **Python 3**, tested with multiple inputs, and follow proper coding standards including comments, meaningful variable names, and error handling.

## 🧠 Topics Covered

- **Python Fundamentals** — Variables, data types, operators, input/output
- **Conditionals & Decision Making** — `if`, `elif`, `else` logic
- **Loops** — `for` and `while` loop patterns and iterations
- **Functions & Modular Programming** — Reusable functions, return values, scope
- **Mathematical Computations** — Arithmetic, factorial, prime numbers, number systems
- **String Manipulation** — Slicing, formatting, palindrome, text analysis
- **Mini Projects** — ATM Simulator, Guessing Game, Function Calculator, and more
